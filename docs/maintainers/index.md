## Maintainer Responsibilities

### Code Review

- Review all incoming PRs within 48 hours
- Ensure code quality and standards
- Provide constructive feedback
- Merge approved PRs

### Community Management

- Monitor Telegram and GitHub discussions
- Answer community questions
- Triage issues and feature requests
- Recognize contributors

### Release Management

- Plan release schedules
- Create release notes
- Tag and publish releases
- Update documentation

## Access Requirements

Maintainers need access to:

- GitHub repository (write access)
- Telegram admin group
- Release management tools

Contact the core team to request access.
