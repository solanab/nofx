# Roadmap

NOFX's strategic development plan and market expansion.

---

## Overview

NOFX is continuously evolving to become the leading AI-powered trading platform. Our roadmap outlines the key milestones and features we're working on.

---

## Short-term Goals (Q2-Q3 2025)

### Advanced Risk Management

- [ ] Dynamic position sizing based on volatility
- [ ] Multi-level stop-loss strategies
- [ ] Portfolio-level risk controls
- [ ] Drawdown protection mechanisms

### Multi-AI Ensemble

- [ ] AI model voting system
- [ ] Weighted ensemble decisions
- [ ] Model performance auto-selection
- [ ] A/B testing framework for strategies

### New Exchange Integrations

- [x] Binance Futures
- [x] Bybit
- [x] OKX
- [x] Hyperliquid
- [x] Aster DEX
- [x] Lighter
- [ ] More exchanges coming...

### Enhanced UI/UX

- [ ] Advanced charting with TradingView integration
- [ ] Mobile-responsive dashboard
- [ ] Custom notification system
- [ ] Strategy templates library
