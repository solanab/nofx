# 路线图

NOFX 的战略发展计划和市场扩展。

---

## 概述

NOFX 正在不断发展，以成为领先的 AI 驱动交易平台。我们的路线图概述了我们正在努力的关键里程碑和功能。

- [ ] 基于波动率的动态仓位管理

## 如何为路线图做贡献

我们欢迎社区对路线图的意见！

1. **建议功能** - 打开 [GitHub Discussion](https://github.com/tinkle-community/nofx/discussions)
2. **为功能投票** - 对现有功能请求进行反应
3. **贡献代码** - 选择路线图项目并实现它们
4. **赞助开发** - 通过悬赏支持特定功能
